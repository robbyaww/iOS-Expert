# iOS Expert

A self-paced iOS development learning app that takes you from absolute beginner to big-tech-interview-ready iOS engineer.

## Overview

iOS Expert is a native iOS app built with SwiftUI that serves as your comprehensive study companion for mastering iOS development. The curriculum is structured into **6 phases** across **30 modules**, each containing theory, code examples, mini projects, and curated learning resources.

## Curriculum

| Phase | Title | Modules | Focus |
|-------|-------|---------|-------|
| 1 | Foundation | 1–6 | Swift language fundamentals & Xcode tooling |
| 2 | SwiftUI & App Architecture | 7–12 | Modern UI, state management, MVVM + TCA |
| 3 | Data, Networking & Persistence | 13–18 | Concurrency, URLSession, SwiftData, Combine |
| 4 | Advanced Topics | 19–24 | Testing, performance, security, CI/CD |
| 5 | System Design & Interview Prep | 25–28 | System design, DSA, behavioral interviews |
| 6 | Capstone Project | 29–30 | Build & ship a production-quality app |

## Features

- **30 Modules** — Each with detailed explanations, topic lists, and code examples
- **Progress Tracking** — Mark modules as complete, track your learning streak
- **Curated References** — YouTube videos, blog posts, Apple docs, Medium articles, courses, and books for every module
- **Big Tech Tips** — Interview-relevant insights for each topic
- **Mini Projects** — Hands-on project ideas to reinforce learning
- **Code Examples** — Syntax-highlighted Swift code snippets with explanations
- **Bookmarks** — Save modules for quick access later
- **Dashboard** — Overview of your learning progress and stats

## Tech Stack

- **Language:** Swift 6
- **UI:** SwiftUI (iOS 17+)
- **Architecture:** MVVM with @Observable
- **IDE:** Xcode 16+

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/robbyaww/iOS-Expert.git
   ```
2. Open `iOS Expert.xcodeproj` in Xcode 16+
3. Build and run on a simulator or device (iOS 17+)
